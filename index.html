Clé API Google Maps

Pour calculer les distances par route (obligatoire depuis 2018 pour le BTP), il faut une clé API Google Maps gratuite.

  1. Aller sur console.cloud.google.com
  2. Créer un projet (ex: "VERIDECO")
  3. Activer Distance Matrix API et Geocoding API
  4. Aller dans "Identifiants" → Créer une clé API
  5. Copier la clé et la coller dans Réglages

200$/mois offerts = largement suffisant pour votre usage.
